Quick start

Fetch trending collections with a single request:

curl https://api.springnft.io/v1/collections/trending \
  -H "Accept: application/json"

Endpoints

Base URL: https://api.springnft.io/v1

GET  /collections/trending
GET  /collections/:id
GET  /collections/:id/listings
POST /orders/bid
POST /orders/list

Authentication

Include your API key in the Authorization header. Keys are available in the developer dashboard.

Authorization: Bearer sk_live_xxxxxxxx
